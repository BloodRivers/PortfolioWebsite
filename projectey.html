Shooter Multiplayer Action Godot AI-Assisted Development
In Progress

Story

The Beginning

Started as a learning project to understand multiplayer networking in Godot Engine. The goal was to create a simple FPS that could be played with friends, but it quickly evolved into something more ambitious.

Current State

What's Working

Core player movement, weapon systems, and networking are implemented. The game is playable in its current state, with room for many more features and polish.

Key Features

What's Planned

Advanced weapon system, featuring knives and throwables. Multiple game modes, and a progression system are all on the roadmap.

Technical Stack

Under the Hood

Built entirely in Godot Engine 4.6.2 with GDScript. Uses Steam for networking and basic server-authoritative architecture.
